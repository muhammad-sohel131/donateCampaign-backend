# DonationCamp Server

This is the server-side application for the **DonationCamp** project, a platform for managing campaigns and donations. The server is built using Node.js, Express.js, and MongoDB for handling RESTful APIs.  

---

## Features

- **Campaign Management**:
  - Create, retrieve, update, and delete campaigns.
  - View specific campaigns by ID.
  - Retrieve campaigns created by a specific user.

- **Donation Management**:
  - Donate to a campaign.
  - View user donations, including campaign details.
  
- **Running Campaigns**:
  - Fetch a list of active campaigns based on the deadline.
